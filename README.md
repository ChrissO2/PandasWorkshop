# Pandas Workshop

A comprehensive pandas learning workshop demonstrating data analysis and visualization techniques using football transfer data.

## Overview

This project provides practical examples of common pandas operations including data cleaning, aggregation, filtering, and visualization. It is designed as an educational resource for learning pandas through real-world data analysis.

## Contents

- **main.ipynb** - Interactive notebook with pandas tutorials and exercises
- **transfer_data.csv** - Sample dataset containing football transfer information

## Dataset

The dataset is sourced from [Kaggle Transfers in 2016/17 Season](https://www.kaggle.com/datasets/vardan95ghazaryan/transfers-in-1617-season) and is included in the repository for easy access.

## Requirements

- Python 3.x
- pandas
- matplotlib
- seaborn

## Usage

Open `main.ipynb` in a Jupyter notebook environment to explore the tutorials and examples.
